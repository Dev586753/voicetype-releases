# VoiceType

Application locale de reconnaissance vocale (speech-to-text) utilisant OpenAI Whisper.
Fonctionne entièrement hors-ligne. Aucun envoi de données.

## Téléchargement

Téléchargez la dernière version depuis la page [Releases](../../releases).
