# VoiceType

**Moteur de transcription vocale en temps réel** — Speech-to-Text Engine

VoiceType transforme votre voix en texte instantanément, directement sur votre machine. Aucune connexion internet requise pour la transcription.

## Téléchargement

Rendez-vous dans les [**Releases**](https://github.com/Dev586753/voicetype-releases/releases/latest) pour télécharger la dernière version.

## Fonctionnalités

- Transcription vocale en temps réel (Whisper)
- Traitement 100% local — aucune donnée envoyée sur internet
- Support multilingue (français, anglais, espagnol, allemand, etc.)
- Raccourci clavier configurable (push-to-talk)
- Authentification Windows Hello / mot de passe
- Monitoring des ressources système
